# SNAKE GAME

Jogo de Snake em linguagem c

# Sobre
Projeto de demostração de utilização dos seguintes recursos 
avançados de programação em linguagem C:

	- Ponteiros
	- Alocação dinâmica
	- Listas duplamente ligadas
	- Ordenação 
	- Threads (sub-processos)
	- etc ...
